# Anka-tool-Vidiyo-indirici
ANKA Video Downloader Tool, Termux üzerinde çalışan yt-dlp tabanlı bir video indirme aracıdır ve Instagram, TikTok, YouTube ve Pinterest videolarını farklı kalite seçenekleriyle indirmenizi sağlar.

Kurulum için gerekli dosyalar ve komutlar:
1) git clone https://github.com/anka71/anka-tool.git
2) cd anka-tool
3) pkg install python -y
4) pkg install termux-api -y
5) pip install -U yt-dlp

Çalıştırma:
python ankavidiyoindirici.py
